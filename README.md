# Atelier - Asistente de IA con RAG & Text2SQL integrado

Atelier es un asistente virtual inteligente avanzado diseñado para boutiques y comercios electrónicos. Combina la recuperación de información no estructurada (como políticas de envío, devoluciones y guías de tallas) con la consulta en tiempo real de bases de datos relacionales (ventas, stock y productos) utilizando una arquitectura híbrida de IA.

La solución integra un backend robusto basado en **Flask**, un sistema de toma de decisiones orquestado por **LangGraph**, y una interfaz de usuario minimalista y elegante construida en **React** con **Vite**.
